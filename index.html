PRACTICA ETIQUETAS

Etiqueta CITE

El elemento "cite" representa a una cita del título de una obra. Este elemento puede ser usado para citar los títulos

Ejemplo

La película Fast And Furious X saldrá en 2023 La película Fast And Furious X saldrá en 2023

Etiqueta Q

El elemento "q" indica que el texto adjunto es una cita corta en línea. Al utilizar este elemnto, el texto se pondra entre comillas.

Ejemplo

La película Fast And Furious X saldrá en 2023 La película Fast And Furious X saldrá en 2023

Etiqueta DFN

El elemento "dfn" sirve para marcar el término que se quiere definir.

Ejemplo

La película Fast And Furious X saldrá en 2023

Etiqueta ABBR

El elemento "abbr" sirve para realizar una abreviación del texto que pongamos, en este caso cojera las mayusculas del texto.

Ejemplo

Yo estudio HTML
